File Organizer (Python)

Overview
A Python automation project that automatically organizes files into folders based on their file extensions.
The program scans a target folder, creates categorized folders, and moves files into their appropriate locations.

Features

* Automatically detects file types
* Creates folders if they do not exist
* Organizes documents, images, and music files
* Uses Python file handling and automation concepts
* Beginner-friendly and easy to extend

Technologies Used

* Python
* os module
* shutil module

Folder Structure

file-organizer-python/

├── organizer.py

├── README.md

└── test_folder/

├── Documents/

├── Images/

└── Music/

Supported File Types

Documents

* .pdf
* .txt
* .docx

Images

* .jpg
  
Music

* .mp3

How to Run

1. Clone the repository
2. Open the project in VS Code
3. Run:

python organizer.py

4. Files will be organized automatically

Learning Outcomes

* Working with files and directories
* Python automation
* Loops and conditional statements
* File organization using Python
* Practical use of os and shutil modules
